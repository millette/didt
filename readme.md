# Did I do that?

## Installation

Requires node 8.9.4 (LTS) or above.

```
git clone ...
yarn # ou npm install
```

Create a ```.env``` file with GITHUB_TOKEN=XXX where XXX is your, you guessed it, github token.

## Run
```
node cli USERNAME > fichier-des-projets.txt
```

Replace USERNAME with your username.
